# 贪吃蛇 (Snake Game)

这是一个使用原生 HTML5 Canvas、CSS 和 JavaScript 编写的简单贪吃蛇游戏。

## 如何运行

1.  确保你已安装任意现代浏览器（Chrome, Firefox, Edge, Safari 等）。
2.  直接双击打开 `snake-game` 目录下的 `index.html` 文件即可开始游戏。
3.  或者，如果你使用 VS Code，可以使用 "Live Server" 插件打开。

## 游戏控制

*   **移动**: 使用键盘 **方向键** (↑ ↓ ← →) 或 **WASD** 键控制蛇的移动方向。
*   **暂停/继续**: 按 **空格键 (Space)** 暂停或继续游戏。
*   **重新开始**: 游戏结束或进行中时，按 **R** 键重新开始。

## 游戏规则

1.  吃掉红色的食物增加分数和蛇的长度。
2.  避免撞到墙壁或蛇自己的身体，否则游戏结束。
3.  游戏会自动记录并在本地存储你的最高分。

## 目录结构

*   `index.html`: 游戏主页面结构。
*   `style.css`: 游戏样式设计。
*   `game.js`: 游戏核心逻辑（渲染、移动、碰撞检测等）。

## 开发

本项目没有任何外部依赖，纯原生代码实现。
